# Currency-Converter
A real-time currency converter web application built with HTML, CSS, and JavaScript using exchange rate APIs and dynamic country flags.
Features
Real-time exchange rates
Dynamic currency dropdowns
Country flag updates
Currency swap functionality
Automatic conversion on input changes
Responsive and modern UI
Error handling for API failures
Tech Stack
HTML
CSS
JavaScript
Fetch API
Async/Await
Concepts Practiced
DOM Manipulation
Event Handling
API Integration
Asynchronous JavaScript
Dynamic UI Updates
